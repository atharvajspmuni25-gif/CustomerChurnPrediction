# CustomerChurnPrediction
A Machine Learning project that predicts whether a customer will stay or leave a company using classification algorithms and a Streamlit web application.
# 📞 Customer Churn Prediction System

## 📌 Project Overview

The Customer Churn Prediction System is a Machine Learning-based web application designed to predict whether a customer is likely to leave a company or continue using its services.

Customer churn prediction is one of the most important business analytics applications used in industries such as:

* Telecommunications
* Banking
* Insurance
* OTT Platforms
* E-commerce

This project uses customer information and service-related features to train a Machine Learning model that can predict churn behavior accurately.

The application is developed with Python, Scikit-learn, and Streamlit, providing an interactive and user-friendly web interface for predictions.

# 📊 Dataset Information

Dataset used:
Telco Customer Churn Dataset

Dataset Source:
https://www.kaggle.com/datasets/blastchar/telco-customer-churn

The dataset contains customer-related information such as:

* Gender
* Senior Citizen Status
* Partner Information
* Dependents
* Tenure
* Phone Services
* Billing Information
* Monthly Charges
* Total Charges

Target Variable:

* Churn (Yes / No)

# ⚙️ Project Workflow

## 1️⃣ Data Collection

The customer churn dataset is collected from Kaggle.

## 2️⃣ Data Preprocessing

* Removed unnecessary columns
* Handled missing values
* Converted categorical data into numerical values using Label Encoding

## 3️⃣ Feature Selection

Important customer-related features are selected for prediction.

## 4️⃣ Model Training

A Decision Tree Classifier is trained using Scikit-learn.

## 5️⃣ Model Evaluation

The model is evaluated using Accuracy Score.

## 6️⃣ Deployment

The trained model is integrated into a Streamlit web application.
